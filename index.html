Telemetry Anomaly Detection

A system that ingests multi-channel telemetry data — speed, throttle, brake pressure, g-forces, tyre temperatures, sector times — and surfaces statistically anomalous behaviour that could indicate mechanical events, driving errors, or setup issues, displayed on a monitoring dashboard.

Post-Race Report Generator

A pipeline that automatically ingests the previous race's data via FastF1 after each Grand Prix weekend and produces a structured, visually rich PDF report — the kind a performance engineer reads on Monday morning.

FastF1 matplotlib Jinja2 WeasyPrint GitHub Actions

Lap Time Delta Predictor

A predictive model for lap time degradation as a function of tyre compound, tyre age, fuel load estimate, track temperature, and circuit characteristics — paired with an interactive dashboard through which an engineer can interrogate predictions for upcoming stints.

XGBoost LightGBM SHAP FastF1 Plotly Dash

Incident Detection

Complete rebuild of a motorsport incident classifier — EfficientNet-B3 replacing a custom CNN, 1000+ image dataset replacing 200, Grad-CAM validation baked into training to eliminate livery label leakage. Deployed as a web app: upload a clip, get a classification and heatmap overlay.

PyTorch EfficientNet-B3 Grad-CAM FastAPI React

A simplified F1 race strategy environment modelling pit stop timing, tyre selection, and safety car response — with a reinforcement learning agent trained to optimise finishing position against rule-based opponent cars.
