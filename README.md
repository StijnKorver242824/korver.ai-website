# Motorsport Portfolio — Stijn Korver

A racetrack-as-timeline portfolio for Stijn Korver, a BSc Applied Data Science & AI student aiming for Formula 1.

**Two views of the same design ship in this project:**

- `index.html` + `projects/` + `apps/` — a self-contained static prototype you can open offline.
- `jekyll/` — a Jekyll source tree ready to drop into a GitHub Pages repo.

Start at **`index.html`** for the live prototype. See **`jekyll/README.md`** for deploy instructions.
