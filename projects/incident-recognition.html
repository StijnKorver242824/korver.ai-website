Open App ↗ Source ↗
/ 01Problem

Race control reacts. Engineers want to anticipate.

Locking a tyre, going off, debris on track — each one has a video signature and a telemetry signature. Fuse them, and a classifier can label the event in seconds.

/ 02Solution

YOLO on broadcast + LSTM on telemetry → ensemble.

YOLOv8 spots cars, flags, and smoke; an LSTM consumes the matching window of telemetry. A late-fusion classifier returns the most likely incident type with calibrated confidences.

[ figure 1 — incident scope ]
/ 03Outcome

Working prototype across 9 incident classes.

Biggest constraint is labelled broadcast footage. Currently expanding the dataset and exploring weak supervision from race-control messages as a free label source.
